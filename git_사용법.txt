git init                - 해당 폴더에서 깃을 사용할 수 있도록 시작합니다
git status              - 현재 git의 상태를 보여줍니다.
git remote add '리모트명' 'URL' - 해당 레보지토리의 리모트를 '리모트명'으로 추가해 줍니다. 개인 작업때는 'origin' 으로 통일합니다.

git clone 'URL'           - 해당 레포지토리의 소스를 복사합니다.

git add .                 - 폴더내의 모든 파일을 add 합니다.
git add '파일명'           - 파일명의 파일을 add 합니다.

git commit -m '메모내용'   - add 상태의 파일들을 커밋 합니다. 해당 커밋에 대한 메모를 할 수 있습니다.

git push origin master     - origin 리모트로 푸시를 진행합니다.

rm -r .git                 - .git 폴더가 삭제되며 git 로컬 저장소 지정을 해제합니다.

git config user.name      - 로그인된 계정 이름 확인
git config user.email     - 로그인된 계정 이메일 확인


git config --global user.name '변경희망이름'     - 연결되어 있는 git 계정을 변경합니다.
git config --global user.email '변경희망이메일'  - 연결되어 있는 git 계정을 변경합니다.
